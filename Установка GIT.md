# Установка GIT

Прежде чем начинать использовать **Git**, нужно сделать его доступным на вашем компьютере. Даже если он уже установлен, вероятно, будет хорошей идеей обновить его до последней версии. Можно либо установить его в виде пакета, либо с помощью другого установщика, либо загрузить исходный код и скомпилировать его самостоятельно.
